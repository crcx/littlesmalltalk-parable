A Little Smalltalk by crcx

A Little Smalltalk, Version 3

Little Smalltalk is a lightweight implementation of Smalltalk created by Tim Budd. This is my fork of the original LSTv3 code with updates and bug fixes to enable it to be used on modern systems.

Install

1. Run 'make' 2. Run ./lst3
